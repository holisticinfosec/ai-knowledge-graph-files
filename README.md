# ai-knowledge-graph-files
Files for experimentation with toolsmith #153: AI-Powered Knowledge Graph Generator &amp; APTs

Holisticinfosec.io's discussion of Robert McDermott's [AI-Powered Knowledge Graph Generator](https://github.com/robert-mcdermott/ai-knowledge-graph) in toolsmith #153 includes text files generated from articles for ingestion into the models, as well as the output files for your use and experimentation should you not wish to generate them on your own.   
Refer to AI-Powered Knowledge Graph Generator & APTs toolsmith #153: unstructured text to interactive knowledge graph via LLM & SPO triplet extraction.
